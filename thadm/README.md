# Thadm Product Images

This folder contains product screenshots for **Thadm** used on the marketing website.

## Images

| File | Description | Used In |
|------|-------------|---------|
| `Thadm_Home_Page.png` | Main timeline view with search | App Preview (Home tab) |
| `Thadm_Tasks_Page.png` | AI-extracted tasks view | App Preview (Tasks tab) |
| `AI_models_Page.png` | Local AI model management | App Preview (AI Models tab) |
| `Audio_Section.png` | Audio recording settings | App Preview (Audio tab) |
| `Privacy_Page.png` | Privacy controls and settings | App Preview (Privacy tab) |
| `Connections_Page.png` | App integrations page | App Preview (Connections tab) |
| `Recordings_Page.png` | Audio recordings with transcripts | Recordings Gallery section |

## Usage

Reference images in React components:

```tsx
<img src="/thadm/Thadm_Home_Page.png" alt="Thadm Home" />
```

## Adding New Images

1. Export screenshots at 2x resolution (recommended: 2880×1800 or higher)
2. Use PNG format for UI screenshots
3. Follow naming: `Feature_Name.png` (PascalCase with underscores)
4. Update this README with the new image details
